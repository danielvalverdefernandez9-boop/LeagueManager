# League Manager

Aplicación de escritorio desarrollada en JavaFX y MySQL para la gestión de competiciones deportivas.

## Autor

-Daniel Valverde Fernández

## Funcionalidades

- Gestión de equipos
- Gestión de jugadores
- Gestión de entrenadores
- Gestión de partidos
- Gestión de competiciones
- Clasificación automática
- CRUD completo con base de datos
